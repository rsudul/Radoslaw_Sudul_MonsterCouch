# Monster Couch Test Project
A small Unity 2D project created as a part of a programming test.
## Overview
The project uses Unity 2022.3.21f1, and consists of three main elements:
1. **Main Menu** - Play, Settings and Exit buttons.
2. **Settings Screen** - A simple UI showing 2 interactable checkboxes.
3. **Play Mode** - A simple game showing player and 1000 enemies as sprites. Player can move using keyboard, enemies move within the screen boundaries and stop moving upon contact with player.
